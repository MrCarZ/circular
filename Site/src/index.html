Sobre o Projeto

É uma proposta desenvolvida pelo PETi (projeto coordenado pelo LASSE - Núcleo de P&D em Telecomunicações, Automação e Eletrônica) que visa atender à população que está diariamente na UFPA e necessita do transporte coletivo “circular”. O ideal seria haver um número maior de ônibus. Mas outro problema enfrentado pelos usuários é a questão do tempo, pois, mesmo estando dentro do campus, a demora para se locomover de um portão para outro é grande e fica difícil planejar ao não se saber onde o ônibus está no momento. É a partir dessa problemática que o projeto surge. A intenção é trazer mais conforto para todos indicando em tempo real a posição dos ônibus.

Agradecimento aos que fizeram este projeto acontecer: Felipe Bastos, Paulo Alberto, Lucas Borges, Izidio Carvalho, Antonio Adrian, Lucas Shibata, Ingrid Nascimento, Moacir Neto, Camila Novaes, Carlos Eduardo Dias, Gabriel Couto, Marcos Lude, Virgínia Tavares e Yuri Silva.
